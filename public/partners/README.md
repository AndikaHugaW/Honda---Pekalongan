## Catatan:

- Pastikan file SVG memiliki format yang benar
- Logo akan otomatis di-invert menjadi putih untuk tampilan di dark background
- Ukuran logo akan disesuaikan otomatis (max 120px width, 60px height)
- Logo akan terlihat dengan opacity 70% dan menjadi 100% saat hover

## Cara Menambahkan Logo:

1. Simpan file SVG dengan nama sesuai daftar di atas
2. Letakkan di folder `public/partners/`
3. Logo akan otomatis muncul di section partner logos
